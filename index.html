Pokemon Sizes!


The api I selected gives all of the information of a pokemon based on the id number or name entered.

I found this API on Medium.com's 15 fun and interesting APIs in 2022. https://medium.com/codex/15-fun-and-interesting-apis-to-use-for-your-next-coding-project-in-2022-86a4ff3a2742

The parameters for the API request are identification methods for pokemon which in this api is either the name or its id number

This API would be helpful in a pokemon quiz and pokemon search application
